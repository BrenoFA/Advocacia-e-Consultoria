# Rosário e Schmitt Advocacia e Consultoria - Landing Page

Uma landing page profissional e moderna desenvolvida para o escritório **Rosário e Schmitt Advocacia e Consultoria**, conduzido por advogadas especializadas em Direito Imobiliário e Sucessório. O design transmite confiança, elegância e segurança, refletindo a seriedade dos serviços prestados.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias web fundamentais, garantindo alta performance, SEO otimizado e fácil manutenção:

- **HTML5**: Estruturação semântica para acessibilidade e SEO.
- **CSS3**: Estilização customizada, utilizando variáveis (Custom Properties), Flexbox, CSS Grid e animações suaves (micro-interações e *reveal on scroll*).
- **JavaScript (ES6+)**: Lógica para interatividade, manipulação do DOM, formulário de contato e *banner* de cookies.

## ✨ Funcionalidades e Destaques

- **Design Responsivo**: Adapta-se perfeitamente a dispositivos móveis, tablets e desktops.
- **Animações (Reveal on Scroll)**: Elementos surgem suavemente na tela à medida que o usuário faz a rolagem, aumentando o engajamento.
- **Formulário de Contato**: Interface clara para captação de leads, com validação de campos e checkbox de consentimento da LGPD.
- **Banner de Cookies**: Aviso de consentimento integrado, reforçando a adequação à Lei Geral de Proteção de Dados (LGPD).
- **SEO Otimizado**: Uso adequado de tags meta, Open Graph e hierarquia de títulos (H1, H2, H3).

## 📂 Estrutura do Projeto

- `index.html`: Arquivo principal contendo a estrutura da página.
- `styles.css`: Arquivo de estilos contendo o Design System (cores, tipografia, espaçamentos) e as regras visuais.
- `script.js`: Arquivo de scripts responsáveis pelas interações e validações.
- `assets/images/`: Diretório contendo as imagens, backgrounds e logotipos utilizados na interface.
